14.2.5 Resource ActivityGroup - Detailed Descriptions

Detailed Descriptions for the elements in the ActivityGroup resource.

ActivityGroup
Definition

A guidance response is the formal response to a guidance request, including any output parameters returned by the evaluation, as well as the description of any proposed actions to be taken.

Control1..1
ActivityGroup.identifier
Definition

Allows a service to provide a unique, business identifier for the response.

NoteThis is a business identifer, not a resource identifier (see discussion)
Control0..1
TypeIdentifier
Summarytrue
ActivityGroup.subject
Definition

The patient for which the request was processed.

Control0..1
TypeReference(Patient | Group)
ActivityGroup.context
Definition

Allows the context of the guidance response to be provided if available. In a service context, this would likely be unavailable.

Control0..1
TypeReference(Encounter | EpisodeOfCare)
ActivityGroup.occurrenceDateTime
Definition

Indicates when the guidance response was processed.

Control0..1
TypedateTime
ActivityGroup.author
Definition

Provides a reference to the device that performed the guidance.

Control0..1
TypeReference(Device | Practitioner)
ActivityGroup.reason[x]
Definition

Indicates the reason the request was initiated. This is typically provided as a parameter to the evaluation and echoed by the service, although for some use cases, such as subscription- or event-based scenarios, it may provide an indication of the cause for the response.

Control0..1
TypeCodeableConcept|Reference(Any)

ActivityGroup.note

Provides a mechanism to communicate additional information about the response.

TypeAnnotation
ActivityGroup.action
Definition

The actions, if any, produced by the evaluation of the artifact.

Control0..*
ActivityGroup.action.actionIdentifier
Definition

A unique identifier for the action. The identifier SHALL be unique within the container in which it appears, and MAY be universally unique.

Control0..1
TypeIdentifier

A user-visible label for the action.

Control0..1
Typestring
ActivityGroup.action.title

The title of the action displayed to a user.

Control0..1
Typestring
ActivityGroup.action.description
Definition

A short description of the action used to provide a summary to display to the user.

Control0..1
Typestring
ActivityGroup.action.textEquivalent
Definition

A text equivalent of the action to be performed. This provides a human-interpretable description of the action when the definition is consumed by a system that may not be capable of interpreting it dynamically.

Control0..1
Typestring
ActivityGroup.action.code

The concept represented by this action or its sub-actions.

Control0..*

ActivityGroup.action.documentation
Definition

Didactic or other informational resources associated with the action that can be provided to the CDS recipient. Information resources can include inline text commentary and links to web resources.

TypeRelatedArtifact
