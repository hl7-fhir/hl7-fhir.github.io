11.2.5 Resource MedicationRequest - Examples

ORAL DOSAGE FORMS

The following examples are MedicationRequest resources for Oral Dosage Forms that reference Medication Resources for generic formulations:

  • Active Order with link to encounter, reasonCode, note multiple dosage lines, dispenseRequest and substitution - Azithromycin
  • Order with as needed (PRN) dosage with pre-condition - Oxycodone - active with reasonCode, note, dispenseRequest and substitution
  • Order for Tapering Dose - Prednisone - active with note and multiple DosageInstructions with boundsPeriod
  • Order for Oral Drops - Nystatin Suspension - completed - with Dispense Request
  • Order with Dosage with a Pre-Condition - Alprazolam - dosageInstructions includes boundsPeriod with periodMax and asNeededCodeableConcept
  • Order for a Dosage that needs to be dispensed as different strength tablets - Warfarin - active

The following examples are MedicationRequest resources for Oral Dosage Forms that reference Medication Resources for manufactured products:

  • Oral Chemotherapy Order - Myleran - completed - with reasonCode
  • Order with as needed (PRN) dosage - Percocet - completed with link to encounter, dispenseRequest and substitution
  • Order for as needed (PRN) dosage - Vicodin - completed with dosageInstruction with additionalInstruction (coded) and as needed with reason, dispenseRequest and substitution
  • Oral Chemotherapy Order - Capecitabine - active medication with dosing by Body Surface Area (BSA) (mg/m2)
  • Order for Over the Counter Medication - Tylenol PM - with dosage range as needed with pre-condition for use
  • Order for a medication that includes the dosage of a prescription in text
  • Fully populated example of a MedicationRequest - Chlorthalidone - active - with link to encounter, reasonReference, note, dispenseRequest, substitution and eventHistory

The following examples are MedicationRequest resources for Oral Dosage Forms that include a Medication Code

  • Liquid Medication - Phenytoin with link to prior prescription
  • Order with a single dosage line - Azithromycin - completed - with link to prior prescription and reason code
  • Order - Levothyroxine - completed with reasonCode, dosageInstructions and dispenseRequest

INJECTABLE DOSAGE FORMS

The following examples of MedicationRequest resources for Injectable Dosage Forms that reference a Medication Resource for a generic formulation

  • Order for an injectable medication - Lorazepam - active - with link to encounter, reasonCode and dosageInstruction

The following examples of MedicationRequest resources for Injectable Dosage Forms that reference a Medication Resource for a manufactured product:

  • Order for Intravenous Chemotherapy (inpatient order) - Brentuximab Vedotin - completed
  • Order for Intravenous Chemotherapy (inpatient order) - Alemtuzumab - completed with multiple dosageInstruction lines
  • Order for Intravenous Antibiotic - Vancomycin - active - with link to encounter, reasonCode, and note
  • Order for Intravenous Antibiotic with a Dosage Rate - Piperacillin - completed with reason code
  • Order for Subcutaneous injection with multiple dosage lines - Insulin - Novolog - active - includes reasonCode and note. DosageInstruction includes additionalInstructions as text

The following examples of MedicationRequest resources for Injectable Dosage Forms that include a Medication Code:

  • Order for Subcutaneous injection - Insulin - Lantus - completed, with reason code and dispense request

The following examples of MedicationRequest resources for Injectable Dosage Forms for Injectable Dosage Forms where the medication is a compounded (aka extemporaneous or magistral) product:

  • Order for Intravenous Admixture - Potassium Chloride in D5W - completed with reason code
  • Order for TPN Solution - completed

MISCELLAENOUS DOSAGE FORMS

The following examples of MedicationRequest resources including miscellaneous dosage forms:

  • Order using Medication Code for Rectal Suppository - Acetaminophen - completed with reasonCode, dispenseRequest and substitution
  • Order using Medication Code for Topical Ointment - Nystatin - on hold and with dispense request
  • Order using Medication Code for Inhaler - Proventil - on hold with dispenseRequest and substitution
  • Order using Medication Code for Patch - Fentanyl - active, with dispense request
  • Medication Resource Reference - Nasal Spray - Nasonex
  • Order referencing Medication Resource for Compounded Product - HC 1%, Salicylic Acid 5% in Glaxal Base - on hold with dispenseRequest
  • Order referencing Medication Resource for Eye Drops - Timoptic - active - with link to encounter, dispenseRequest and substitution flag
