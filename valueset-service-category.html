4.3.1.879 Value Set http://hl7.org/fhir/ValueSet/service-category

This is a value set defined by the FHIR project.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/service-category
Name:ServiceCategory
Definition:This value set defines an example set of codes that can be used to classify groupings of service-types/specialties.
Committee:Patient Administration Work Group
OID:2.16.840.1.113883.4.642.2.879 (for OID based terminology systems)
