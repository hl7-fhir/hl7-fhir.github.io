4.2.0.3.1 ICD-9

The URL "http://hl7.org/fhir/sid/icd-9" is the FHIR system identifier for codes defined in the ICD-9 US code system. The source of this code system is the CDC, at http://www.cdc.gov/nchs/icd/icd9.htm .
