4.3.2.60 HL7 v3 Code System LanguageAbilityMode

This code system (http://hl7.org/fhir/v3/LanguageAbilityMode) is defined as part of HL7 v3.

Summary

Defining URL:http://hl7.org/fhir/v3/LanguageAbilityMode
Name:v3 Code System LanguageAbilityMode
Definition: A value representing the method of expression of the language. Example: Expressed spoken, expressed written, expressed signed, received spoken, received written, received signed. OpenIssue: Description copied from Concept Domain of same name. Must be verified.
