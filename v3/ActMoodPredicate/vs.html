4.3.2.10202 HL7 v3 Value Set ActMoodPredicate

This value set (http://hl7.org/fhir/ValueSet/v3-ActMoodPredicate) is defined as part of HL7 v3. Related FHIR content: ActMoodPredicate.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/v3-ActMoodPredicate
Name:ActMoodPredicate
Definition: Any of the above service moods (e.g., event, intent, or goal) can be turned into a predicate used as a criterion to express conditionals (or queries.) However, currently we allow only criteria on service events.
OID:2.16.840.1.113883.1.11.10202 (for OID based terminology systems)
