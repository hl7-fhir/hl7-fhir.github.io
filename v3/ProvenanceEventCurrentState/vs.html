4.3.2.20547 HL7 v3 Value Set ProvenanceEventCurrentState

This value set (http://hl7.org/fhir/ValueSet/v3-ProvenanceEventCurrentState) is defined as part of HL7 v3. Related FHIR content: ProvenanceEventCurrentState.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/v3-ProvenanceEventCurrentState
Name:ProvenanceEventCurrentState
Definition: Specifies the state change of a target Act, such as a document or an entry, from its previous state as a predecessor Act. For example, if the target Act is the result of a predecessor Act being "obsoleted" and replaced with the target Act, the source ProvenanceEventCurrentState Act code would be "obsoleted".
