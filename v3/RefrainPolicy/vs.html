4.3.2.20446 HL7 v3 Value Set RefrainPolicy

This value set (http://hl7.org/fhir/ValueSet/v3-RefrainPolicy) is defined as part of HL7 v3. Related FHIR content: RefrainPolicy.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/v3-RefrainPolicy
Name:RefrainPolicy
Definition: Conveys prohibited actions which an information custodian, receiver, or user is not permitted to perform unless otherwise authorized or permitted under specified circumstances. Examples: prohibit redisclosure without consent directive
OID:2.16.840.1.113883.1.11.20446 (for OID based terminology systems)
