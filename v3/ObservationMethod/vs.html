4.3.2.84 HL7 v3 Value Set ObservationMethod

This value set (http://hl7.org/fhir/ValueSet/v3-ObservationMethod) is defined as part of HL7 v3.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/v3-ObservationMethod
Name:v3 Code System ObservationMethod
Definition: A code that provides additional detail about the means or technique used to ascertain the observation. Examples: Blood pressure measurement method: arterial puncture vs. sphygmomanometer (Riva-Rocci), sitting vs. supine position, etc. OpenIssue: Description copied from Concept Domain of same name. Must be verified. Note that the Domain has a full discussion about use of the attribute and constraining that is not appropriate for the code system description. Needs to be improved.
OID:2.16.840.1.113883.1.11.20423 (for OID based terminology systems)
