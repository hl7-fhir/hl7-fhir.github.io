11.1.5 Resource Medication - Examples

The following are examples of Medication resources where the medication is a manufactured product:

  • IV injection (inpatient order) - Manufactured Product - Vancomycin
  • IV Injection - Manufactured Product - Piperacillin
  • IV Chemotherapy Example - Manufactured Product - Alemtuzumab
  • Oral Chemotherapy Example - Manufactured Product - Myleran
  • Eye Drop Example - Manufactured Product - Timoptic
  • IV Chemotherapy Example - Manufactured Product - Adcetris
  • Subcutaneous injection Example - Manufactured Product - Novolog
  • As needed Oral tablet - Manufactured Product - Percocet
  • Over the Counter Product - Manufactured Product - Tylenol PM
  • Oral Chemotherapy Example - Manufactured Product - Xeloda - includes a reference to a substance in ingredient element

The following are examples of Medication resources where the medication is a generic formulation (ie, represents an ingredient or set of ingredients plus dosage form)

  • Generic Formulation - Oxycodone
  • Generic Formulation - Prednisone 5mg tablet
  • Generic Formulation - Nystatin 100,000u/ml
  • Generic Formulation - Lorazepam 2mg/ml injection
  • Generic Formulation - Alprazolam 0.25mg tablet
  • Generic Formulation - Mometasone Furorate 0.05mg/Actuat
  • Generic Formulation - Fully populated example of a Medication - Chlorthalidone
  • Generic Formulation - Oral Chemotherapy Example - Capecitabine – includes a reference to a substance in the Ingredient element
  • Generic Formulation - Azithromycin 250mg oral capsule - includes a reference to a substance in the Ingredient element

The following are examples of Medication resources where the medication is a compounded (aka extemporaneous or magistral) product

  • Compounded Preparation - KCl in D5W Admixture - includes a reference to a substance in the ingredient element
  • Compounded Preparation - TPN Solution - includes a reference to a substance in the ingredient element
  • Compounded Preparation - HC 1%, Salicylic Acid 5% in Glaxal Base - includes a reference to a substance in the ingredient element

The following is an example where the medication resource that is minimally populated

  • Name Element Only populated - Amoxicillin 250mg/5ml Suspension
