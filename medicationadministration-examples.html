11.3.6 Resource MedicationAdministration - Examples

The following are examples of MedicationAdministration resources:

INJECTABLE DOSAGE FORMS

The following examples of MedicationAdministration resources for Injectable Dosage Forms that reference a Medication Resource for a manufactured product:

  • Intravenous Antibiotic - Vancomycin
  • Intravenous Antibiotic with a Dosage Rate - Piperacillin
  • Intravenous Chemotherapy (Inpatient order) - Alemtuzumab - completed
  • Intravenous Chemotherapy (Inpatient order) - Brentuximab Vedotin
  • Example of an IV administration not given with a note - Alemtuzumab

The following examples of MedicationAdministration resources for Injectable Dosage Forms that reference a Medication Resource for a generic formulation:

  • Oral Antibiotic - Azithromycin
  • Lorazepam with a Device

The following examples of MedicationDispense resources for Injectable Dosage Forms that include a Medication Code:

  • Subcutaneous Injection - Insulin - Lantus

The following examples of MedicationDispense resources for Injectable Dosage Forms for Injectable Dosage Forms where the medication is a compounded (aka extemporaneous or magistral) product:

  • Intravenous Admixture - Potassium Chloride in D5W
  • TPN Solution with a Device

ORAL DOSAGE FORMS

The following examples are MedicationDispense resources for Oral Dosage Forms that reference Medication Resources for manufactured products:

  • Oral Chemotherapy - Myleran
  • Dosage with a Pre-Condition - Alprazolam

MISCELLANEOUS DOSAGE FORMS

The following examples of MedicationDispense resources including miscellaneous dosage forms:

  • Medication Code - Patch - Fentanyl
  • Medication Code - Rectal Suppository - Acetaminophen
